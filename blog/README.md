# Blog Folder

This folder is currently empty and ready for future Orkait blog posts.

## Previous Content

The previous blog posts were orkait/GraphQL technical content and have been removed as they're not relevant to Orkait's services.

## Adding New Blog Posts

When you're ready to add blog posts:

1. Create markdown files in this folder with the format: `YYYY-MM-DD-title.md`
2. Uncomment the blog plugin in `docusaurus.config.ts`
3. Uncomment the blog link in the navbar
4. Add blog link back to footer if needed

Example blog post format:
```markdown
---
title: Your Blog Post Title
description: Brief description
authors: author-name
---

Your blog content here...
```
